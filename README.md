# **LunaSpring**
**LunaSpring** - многофункциональная библиотека, облегчающая созданиее плагинов на Bukkit/Spigot/Paper. Она добавляет надёжную систему работы с графическими меню в виде инвентарей с полной свободой действий, упрощает управление конфигурациями, предоставляет удобную систему команд, их автозаполнения, также синхронную и асинхронную работу с базами данных, и большое множество вспомогательных методов. Все для людей)

**Зависимость pom.xml (Maven):**
```xml
<dependency>
      <groupId>com.github.NovaSparkle</groupId>
      <artifactId>LunaSpring</artifactId>
      <version>3.6.15</version>
      <scope>provided</scope>
</dependency>
```
```xml
<repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
</repository>
```
